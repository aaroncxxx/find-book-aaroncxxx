# 出版书籍查找 Find a Book

根据用户需求，定位权威出版版本并提供完整的版本溯源信息。

## 核心流程

```
用户需求 → 识别书籍 → 匹配权威来源 → 输出版本清单(含ISBN)
```

### Step 1: 识别目标

解析用户输入，提取：
- 书名（中/英/德/其他语言）
- 作者/编者/译者
- 具体需求：找版本？查ISBN？对比译本？推荐入门版本？

### Step 2: 匹配权威来源

按优先级查找：
1. **内置数据库** — `references/` 下的权威清单（尼采、中国经典等）
2. **网络搜索** — 使用 mimo_web_search 查找出版社官网、WorldCat、豆瓣读书
3. **ISBN 查询** — 通过 ISBN 搜索工具或 API 验证

### Step 3: 输出格式

```
📚 《书名》权威版本清单

🇩🇪 原版/底本
- 版本名 | 出版社 | 年份 | ISBN: xxx

🇬🇧 英语译本
- 译者 | 出版社 | 年份 | ISBN: xxx

🇨🇳 中文译本
- 译者 | 出版社 | 年份 | ISBN: xxx | 特点: xxx

💡 推荐：...
```

## 查找策略

### 已知书名

1. 检查 `references/` 内置数据库
2. 未命中 → mimo_web_search 搜索 "书名 + 权威版本 + ISBN"
3. 验证 ISBN 有效性（格式校验 + WorldCat 查询）

### 主题式查找

用户说"找关于XX的书" →
1. 确定领域（哲学/文学/宗教/历史）
2. 推荐该领域公认经典
3. 提供每个经典的权威版本清单

### 版本对比

用户问"哪个译本好" →
从四个维度评估：
- 学术权威性（底本来源、译者背景）
- 完整度（是否全译、有无删节）
- 可读性（语言风格、注释质量）
- 获取难度（是否绝版、价格）

## 网络搜索模板

```
# 查找权威版本
"{书名}" authoritative edition ISBN publisher

# 查找中文译本
"{书名}" 中文译本 权威 ISBN 出版社

# 验证 ISBN
site:worldcat.org ISBN {isbn}

# 查找绝版/稀缺版本
"{书名}" "{译者}" 旧书 孔夫子 多抓鱼
```

## 注意事项

- 只提供合法出版物信息，不提供下载链接
- 标注"待核实"的 ISBN 需用户自行验证
- 古籍版本差异大，需说明底本来源
- 翻译质量为主观评估，标注"仅供参考"
